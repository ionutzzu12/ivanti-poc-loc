Ivanti Neurons for MDM permite la creación de configuraciones de aplicación de actualizaciones de software y define el comportamiento de las actualizaciones para el usuario final en cuanto a cadencia y aplazamientos.

Disponible para:

- iOS 18.0 supervisado hasta la versión más reciente compatible con Ivanti Neurons for MDM.
- macOS 15.0 supervisado hasta la versión más reciente compatible con Ivanti Neurons for MDM.

**Procedimiento**

1. Vaya a **Configuraciones** > **+Añadir**.
2. Buscar y seleccionar la configuración de **Ajustes de actualización de Safari**.
3. Configure los ajustes de **Ajustes de actualización de Safari** según la tabla siguiente:| **Ajuste**                                           | **Descripción**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
   \| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   \| Nombre                                               | Introduzca un nombre que identifique a esta configuración.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
   \| Descripción                                          | Introduzca una descripción que explique la finalidad de esta configuración.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
   \| **iOS 18+ supervisado** y **macOS15+**               |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
   \| Inscripción Beta                                     |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
   \| Inscripción al programa (opcional)                   | Seleccione una de las siguientes opciones:\* **Permitido**: el usuario puede inscribirse en cualquier programa beta aplicable asociado con su cuenta registrada en Apple.
   - **Siempre encendido**: se utilizan los programas beta especificados por la organización, y el usuario no puede inscribirse en un programa beta utilizando su cuenta registrada en Apple.**Programa de ofertas**: Con el Programa de ofertas, puede hacer cumplir un programa requerido. Al aplicarlo, puede establecer las versiones requeridas para dispositivos iOS y macOS.**Requerir programa**: el dispositivo se inscribe automáticamente en este programa beta.
   - **Siempre apagado**: el dispositivo no puede inscribirse en ningún programa beta. Si ya está inscrito, el sistema elimina el dispositivo de cualquier programa beta. |
     \| **iOS 18 supervisado** y **macOS15 supervisado**     |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Notificaciones                                       | Seleccione la casilla de verificación **Notificaciones** y la casilla de verificación **Mostrar todas las notificaciones de aplicación de actualizaciones de software** para mostrar todas las notificaciones de aplicación de actualizaciones de software en el dispositivo del usuario.Si no selecciona las casillas de verificación, el dispositivo muestra las notificaciones activadas una hora antes del plazo de ejecución y reinicia las notificaciones de cuenta atrás.                                                                                                                                                                                                                                                                                                                                         |
     \| Acciones automáticas                                 |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Descargar                                            | Seleccione una de las siguientes opciones:\* **Permitido**: el usuario puede activar o desactivar las descargas automáticas.
   - **Siempre activo**: las descargas automáticas están siempre "Activas" para el usuario.
   - **Siempre inactivo**: las descargas automáticas están siempre "Inactivas" para el usuario.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
     \| Instalar actualizaciones de SO                       | Seleccione una de las siguientes opciones:\* **Permitido**: el usuario puede activar o desactivar para instalar automáticamente las actualizaciones del SO.
   - **Siempre activado**: la instalación automática de actualizaciones del sistema operativo siempre está "activada" para el usuario.
   - **Siempre desactivado**: la instalación automática de actualizaciones del sistema operativo siempre está "desactivada" para el usuario.                                                                                                                                                                                                                                                                                                                                                                                 |
     \| Respuesta de seguridad rápida                        |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Respuesta de seguridad rápida                        | Seleccione la casilla de verificación **Respuesta de seguridad rápida** para controlar la instalación y reversión de actualizaciones. Seleccione las siguientes opciones:\* **Activar**: si se selecciona, ofrece al usuario la instalación de **Respuesta de seguridad rápida**.
   - **Activar reversión**: si se selecciona, ofrece al usuario la reversión de **Respuesta de seguridad rápida**.                                                                                                                                                                                                                                                                                                                                                                                                                         |
     \| **iOS 18+ supervisado**                              |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Cadencia recomendada                                 | Seleccione una de las opciones siguientes para mostrar las actualizaciones de software al usuario cuando haya más de una actualización disponible:\* **Todas**: muestra todas las versiones de actualización del software.
   - **Más antigua**: muestra solo la versión de actualización de software más antigua (con el número más bajo).
   - **Más nueva**: muestra solo la versión de actualización de software más nueva (con el número más alto).                                                                                                                                                                                                                                                                                                                                                                        |
     \| Aplazamientos                                        | Puede optar por posponer las actualizaciones de software para todos los dispositivos supervisados.No se puede posponer la **Respuestas de seguridad rápidas**.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
     \| Periodo combinado en días                            | Establezca los días para posponer una actualización del sistema principal o secundaria en el dispositivo. Una vez configurada, la actualización de software principal o secundario aparecerá transcurrido un número determinado de días desde su publicación.Puede establecer cualquier valor entre 1 y 90 días.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
     \| **macOS 15+ supervisado**                            |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Permitir actualizaciones estándar del SO del usuario | Seleccione la casilla de verificación **Permitir actualizaciones estándar del SO del usuario** para permitir que un usuario realice actualizaciones del SO.Seleccione la casilla de verificación **Permitir al usuario estándar realizar actualizaciones de software principales y secundarias&#x20;**&#x70;ara permitir que un usuario estándar realice actualizaciones de software principales y secundarias.Solo los administradores pueden realizar actualizaciones de software principales y secundarias si no selecciona la casilla de verificación.                                                                                                                                                                                                                                                               |
     \| Aplazamientos                                        |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Periodo de actualización principal en días           | Establezca los días para posponer una actualización principal del software del sistema operativo en el dispositivo. Una vez configurada, la actualización de software principal aparecerá transcurrido un número determinado de días desde su publicación.Puede establecer cualquier valor entre 1 y 90 días.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
     \| Periodo de actualización secundario en días          | Establezca los días para posponer una actualización secundaria del software del sistema operativo en el dispositivo. Una vez configurada, la actualización de software principal aparecerá transcurrido un número determinado de días desde su publicación.Puede establecer cualquier valor entre 1 y 90 días.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
     \| Periodo de actualización del sistema en días         | Establezca los días para posponer una actualización del sistema SO en el dispositivo. Una vez configurada, la actualización de software principal aparecerá transcurrido un número determinado de días desde su publicación.Puede establecer cualquier valor entre 1 y 90 días.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Acciones automáticas                                 |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
     \| Instalar actualizaciones de seguridad                | Seleccione una de las siguientes opciones:\* **Permitido**: el usuario puede activar o desactivar la instalación de actualizaciones de seguridad.
   - **Siempre activado**: la instalación de actualizaciones de seguridad siempre está "activada" para el usuario.
   - **Siempre desactivado**: la instalación de actualizaciones de seguridad siempre está "desactivada" para el usuario.                                                                                                                                                                                                                                                                                                                                                                                                                                   |
4. Haga clic en **Siguiente** para configurar los ajustes de distribución.
5. Seleccione una de las opciones de distribución para establecer la configuración **Ajustes de actualización de software**. Para más información sobre la configuración de las opciones de distribución, consulte [**Trabajar con configuraciones**](./Trabajar_con_configuraciones.md).
6. Haga clic en **Hecho**.
