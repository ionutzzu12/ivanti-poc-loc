La inscripción móvil de Samsung Knox permite a los administradores registrar dispositivos Samsung cualificados en Ivanti Neurons for MDM. Mediante la Inscripción móvil de Knox, se puede enviar un dispositivo directamente desde un vendedor autorizado a un usuario final, y el cliente de Go Android se descargará automáticamente con los datos de inscripción precargados. Para obtener más información, consulte [**Inscripción móvil de Samsung Knox para Android Enterprise**](https://forums.ivanti.com/s/article/Samsung-KNOX-Mobile-Enrollment-with-MobileIron-Quick-Start-Guide-4666).

### Requisitos

- Lista de dispositivos por IMEI
- Archivo CSV que contiene una lista de dispositivos con un IMEI o número de serie y, opcionalmente, un nombre de usuario y una contraseña de inscripción.
- Neuronas Ivanti for MDM (versión actual).
- Cuenta Samsung Knox aprobada para la inscripción móvil
- Dispositivos compatibles de Samsung: La lista de dispositivos Samsung compatibles está disponible en [**aquí**](https://www.samsungknox.com/en/products/knox-mobile-enrollment).
