Ivanti Neurons for MDM permite a la carga útil de configuración de Red privada móvil proporcionar información del dispositivo sobre los despliegues de redes privadas, incluida la ubicación geográfica, la preferencia sobre Wi-Fi y el tipo de despliegue de la red.

**Aplicable a**

iOS 17.0 hasta la versión más reciente compatible con Ivanti Neurons for MDM.

**Procedimiento**

1. Vaya a **Configuraciones** > **+Añadir**.
2. Busque y seleccione la configuración de **Red de móvil privada**.
3. Configure los ajustes de **Red móvil privada** según la tabla siguiente:| **Ajuste**                       | **Descripción**                                                                                                                                                                      |
   \| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   \| **Nombre**,                      | Introduzca un nombre que identifique a esta configuración.                                                                                                                           |
   \| **Descripción**                  | Introduzca una descripción que explique la finalidad de esta configuración.                                                                                                          |
   \| **Datos móviles preferidos**     | Seleccione la casilla para activar y utilizar los datos móviles a través de Wi-Fi.                                                                                                   |
   \| **NR Independiente**             | Seleccione la casilla si la red de datos móviles tiene NR 5G Standalone.                                                                                                             |
   \| **Nombre del conjunto de datos** | Especifique el Nombre del Conjunto de datos que se identifica con esta configuración.                                                                                                |
   \| **Número de versión**            | Especifique el número de versión del conjunto de datos para realizar un seguimiento de las actualizaciones del sistema.                                                              |
   \| **Geovallas**                    | Especifique la Latitud, Longitud y Radio de cada Geo ID para desplegar una red privada en una ubicación geográfica.Puede crear una lista de hasta mil geovallas para redes privadas. |
4. Haga clic en **Siguiente** para configurar los ajustes de distribución.
5. Seleccione una de las opciones de distribución para establecer la configuración de **Red móvil privada**. Para más información sobre la configuración de las opciones de distribución, consulte [**Trabajar con configuraciones**](./Trabajar_con_configuraciones.md).
6. Haga clic en **Hecho**.
