**Procedimiento**

1. Inicie sesión en Ivanti Neurons for MDM.
2. Vaya a **Configuraciones** > **Agregar configuraciones**.
3. Haga clic (o seleccione) **Ventana de inicio de sesión**.
4. En el campo **Nombre**, introduzca un nombre para la configuración de la ventana de inicio de sesión.
5. Haga clic en **+ Agregar descripción** para agregar detalles.
6. En la sección **Ajustes de configuración**, proporcione los campos siguientes:\* **Información del host de administrador**: introduzca el nombre de host, la dirección IP o SystemVersion.
   - **Lista permitidos**: lista de GUID del usuario o GUID de grupos de usuarios a los que el sistema permite iniciar sesión. Una cadena de asteriscos ('\*') especifica a todos los usuarios o grupos.
   - **Contraseña de inicio de sesión automático**: introduzca la contraseña para Inicio de sesión automático.
   - **Lista no permitido**: la lista de GUID de los usuarios o GUID de grupos de usuarios a los que el sistema no permite iniciar sesión. Esta lista tiene prioridad sobre la lista de la tecla 'AllowList'.
   - **Texto de la ventana de inicio de sesión**: introduzca el texto de la ventana de inicio de sesión.
   - Seleccione la casilla de verificación para opciones adicionales como:\* **Desactivar el acceso a la consola**
     - **Desactivar Inicio de sesión automático de FDE (cifrado de disco de FileVault)**
     - **Desactivar las funciones de bloqueo de pantalla inmediata**
     - **Ocultar a los usuarios del administrador al mostrar una lista de usuarios**
     - **Mostrar solo usuarios de red y sistema**
     - **Ocultar cuentas móviles**
     - **Mostrar usuarios de red**
     - **Desactiva el elemento del menú de inicio de sesión**
     - **Deshabilite el apagado cuando el usuario inicia sesión**
     - **Desactive el elemento del menú de reinicio cuando el usuario inicie sesión**
     - **Desactive el elemento de reinicio**
     - **Mostrar el cuadro de diálogo Nombre y contraseña**
     - Mostrar otros usuarios: administrado
     - Mostrar menú de entrada
     - Desactive el botón de apagado
     - Desactivar el botón de suspensión
     - Desactive el botón del menú de apagado
7. Haga clic en **Hecho**.
