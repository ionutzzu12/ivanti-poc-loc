Una de las primeras cosas que se configuran en Ivanti Neurons for MDM (con el asistente de inicio) es la configuración del código de acceso. Esta configuración define los ajustes para la función de bloqueo de pantalla en los dispositivos.

## Ajustes del código de acceso

| **Ajuste**                                                                             | **Qué hacer**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| -------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nombre                                                                                 | Introduzca un nombre que identifique a esta configuración.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Descripción                                                                            | Introduzca una descripción que explique la finalidad de esta configuración.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Permitir valores simples                                                               | Restringe si el PIN o la contraseña contienen caracteres o dígitos ordenados.**Para iOS, macOS, watchOS, y Android:** seleccione esta opción para permitir PIN o códigos de acceso que sean menos seguros porque contengan secuencias de caracteres repetidas, ascendentes o descendentes.Ejemplos: 1111, 1234, abcd.Si deselecciona esta opción para dispositivos Android, se ejecutarán los códigos de acceso con PIN complejos. Por ejemplo, los usuarios no podrán configurar secuencias de caracteres repetidos, ascendentes o descendentes.**Para Windows 10+**: seleccione permitir códigos de acceso menos seguros porque contengan secuencias de caracteres repetidos o secuencias numéricas ascendentes.Ejemplos: 1111, 1234. |
| Requerir valor alfanumérico                                                            | Requiere que el código de acceso contenga al menos una letra y un número.**Para iOS y Android:** seleccione esta opción para garantizar que los códigos de acceso incluyan letras y números.**Para Windows 10+**: seleccione esta opción para garantizar una contraseña segura basada en el estándar de Microsoft.                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Longitud mínima del código de acceso                                                   | Seleccione un número de la lista para establecer la longitud mínima del código de acceso.**Para Windows 10 Desktop**: las cuentas locales requerirán un código de acceso con una longitud mínima de 6 dígitos.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Número mínimo de caracteres complejos                                                  | **Para iOS y Android**: seleccione un número de la lista para establecer un número mínimo de caracteres que no sean números o letras.**Para Windows 10+**: las cuentas locales aplicarán 3 caracteres complejos.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Antigüedad máxima del código de acceso                                                 | Introduzca un número para la cantidad de días después de los que el usuario del dispositivo debe restablecer el código de acceso. Si no desea establecer la antigüedad del código de acceso, deje este campo en blanco.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| Autobloqueo                                                                            | Seleccione un intervalo de la lista para definir cuánto tiempo puede permanecer inactivo el dispositivo antes de establecer automáticamente el bloqueo de pantalla.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Cualquier método de bloqueo                                                            | **Solo Android**. Permite que el usuario elija cualquier método de bloqueo, incluido el desbloqueo mediante patrón. Los ajustes del código de acceso anteriores no se aplicarán a este dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Historial del código de acceso                                                         | Introduzca un número para establecer la cantidad de códigos de acceso exclusivos que debe introducir el usuario antes de volver a usar un código de acceso. Por ejemplo, si define este campo en 4, el usuario deberá establecer 4 códigos de acceso antes de poder volver a usar el primer código de acceso.                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Período de gracia del bloqueo del dispositivo                                          | Seleccione un intervalo de la lista para establecer el tiempo que pasará desde que aparezca la pantalla de bloqueo y hasta que el usuario del dispositivo tenga que introducir un código de acceso para desbloquear el dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Número máximo de intentos erróneos                                                     | Seleccione un número de la lista para establecer la cantidad de veces que el usuario del dispositivo puede introducir de forma consecutiva el código de acceso incorrecto antes de que el dispositivo se restablezca y se borre.**Advertencia:** los dispositivos se borrarán si el usuario excede la cantidad máxima de intentos de contraseña. Tenga cuidado con esta opción.                                                                                                                                                                                                                                                                                                                                                         |
| (Solo macOS)Activar la expresión regular del código de acceso (macOS 14+)              | Especifique la cadena de expresión que coincide con la contraseña para determinar si coincide con la política.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| (Solo macOS)Idioma                                                                     | Especifique el idioma de la Descripción.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| (Solo macOS)Descripción                                                                | Describir la complejidad de la contraseña Por ejemplo, números, caracteres especiales, cadena, etc.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| (Solo macOS)Aplicar una regla de código de acceso en el siguiente inicio de sesión     | Seleccione esta opción para que macOS solicite al usuario que cambie la contraseña la próxima vez que inicie sesión para que cumpla con la políticas sobre contraseñas.Esta opción no está seleccionada de forma predeterminada.Aplicable a macOS 10.13 y versiones posteriores.                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| (Solo macOS)Minutos hasta el reinicio tras un inicio de sesión fallido                 | Especifique los minutos para que se restablezca el inicio de sesión después de que se haya superado el número máximo de intentos de inicio de sesión fallidos.Asegúrese de que el número máximo de intentos fallidos está establecido para habilitar este campo. Disponible en macOS 10.10 y posterior.                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| SmartLock                                                                              | Para dispositivos Android 5.0 excepto en perfiles de trabajo con la versión corporativa de Android:\*\*Para Android 6.0 o posterior:\*\*permite o no permite a un usuario elegir la función SmartLock para desbloquear un dispositivo. La función SmartLock desbloquea automáticamente un dispositivo en ciertas circunstancias como en la proximidad del usuario con el dispositivo, el dispositivo en una ubicación o cuando el dispositivo se sincroniza con un dispositivo de confianza.                                                                                                                                                                                                                                            |
| Desbloqueo de huella digital                                                           | Para dispositivos Android 5.0 excepto en perfiles de trabajo con la versión corporativa de Android:\*\*Para Android 6.0 o posterior:\*\*Permite o no permite a un usuario elegir la función de huella digital para desbloquear un dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Notificaciones de la pantalla de bloqueo (solo para la versión corporativa de Android) | Permitir notificaciones para los dispositivos administrados en el trabajo (para el propietario del dispositivo)Permitir o no permitir notificaciones en la pantalla de bloqueo para dispositivos administrados del trabajoPermitir notificaciones sin editar para el perfil profesional\*\*Para Android 6.0 o posterior:\*\*Permitir o no permitir notificaciones sin editar en la pantalla de bloqueo para dispositivos administrados del trabajo.Después de activar este ajuste, recibirá la notificación pero el contenido aparecerá como «Contenido oculto por la política». Solo podrá ver el contenido (correo electrónico/notificación push) desde la aplicación.                                                                |
