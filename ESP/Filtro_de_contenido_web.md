La configuración de un filtro de contenido web limita el acceso web de dispositivos iOS 7+, macOS 10.15+ y visionOS 1.1+.

## Ajustes del filtro de contenido web

| **Ajuste**            | **Qué hacer**                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nombre                | Introduzca un nombre que identifique a esta configuración.                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Descripción           | Introduzca una descripción que explique la finalidad de esta configuración.                                                                                                                                                                                                                                                                                                                                                                                                         |
| Sitios web permitidos | **Contenido limitado a adultos:** Seleccione esta opción si desea bloquear el acceso a sitios web basado en filtros automáticos iOS. Estos filtros intentan, con un alto grado de precisión, bloquear sitios web con contenido inadecuado.**Solo sitios web específicos:** Seleccione esta opción si desea enumerar manualmente los sitios web accesibles.**Complemento (solo iOS 8 supervisado y macOS 10.15+):** seleccione esta opción para utilizar un complemento de terceros. |
| URL permitidas        | Esta opción solamente está disponible si ha seleccionado Contenido limitado a adultos.Introduzca las URL permitidas. Cada URL debe comenzar con alguna de estas dos opciones:\* http\://                                                                                                                                                                                                                                                                                            |

- https\://Si desea permitir tanto http\:// como https\:// para el mismo sitio, incluya dos URL separadas.Estarán accesibles todas las URL en las que los caracteres iniciales coincidan con la URL permitida.Por ejemplo: [**http://www.SitioDeMiEmpresa.com**](http://www.SitioDeMiEmpresa.com) permite acceder a las siguientes páginas:\* [**http://www.SitioDeMiEmpresa.com**](http://www.SitioDeMiEmpresa.com)
- [**http://www.SitioDeMiEmpresa.com/empleoSe**](http://www.SitioDeMiEmpresa.com/empleoSe) puede acceder a estas URL incluso si los filtros automáticos de iOS las bloquean.                            |
  \| Use Rechazar las URL de la lista               | Esta opción solamente está disponible si ha seleccionado Contenido limitado a adultos.Introduzca las URL de la lista de bloqueados. Cada URL debe comenzar con alguna de estas dos opciones:\* http\://
- https\://Si desea bloquear http\:// como https\:// para el mismo sitio, incluya dos URL separadas.Estarán bloqueadas todas las URL en las que los caracteres iniciales coincidan con la URL de la lista de bloqueados.Por ejemplo: [**http://www.SitioDeMiEmpresa.com**](http://www.SitioDeMiEmpresa.com) bloquea el acceso a las siguientes páginas:\* [**http://www.SitioDeMiEmpresa.com**](http://www.SitioDeMiEmpresa.com)
- [**http://www.SitioDeMiEmpresa.com/empleoEstas**](http://www.SitioDeMiEmpresa.com/empleoEstas) URL estarán bloqueadas incluso si los filtros automáticos de iOS las permiten. |
  \| Marcadores incluidos en la lista de permitidos | Esta opción solamente está disponible si ha seleccionado Solo sitios web específicos.También puede introducir la carpeta en la que debe añadirse el marcador en Safari.Ejemplo:/Ventas/Productos/Si no existiera, el marcador se añadirá al directorio predeterminado de marcadores.                                                                                                                                                                                                                                                                                                                                                                                                        |
  \| Nombre del filtro                              | Esta opción solamente está disponible si ha seleccionado Complemento.Introduzca el texto que se va a mostrar para identificar este filtro.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
  \| Identificador                                  | Esta opción solamente está disponible si ha seleccionado Complemento.Introduzca el Id. del paquete del complemento que proporciona el servicio de filtros.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
  \| Dirección del servicio                         | Esta opción solamente está disponible si ha seleccionado Complemento.Opcional: Introduzca cualquier dirección del servido necesaria para usar el complemento. Consulte la documentación del complemento para determinar si este valor es necesario.                                                                                                                                                                                                                                                                                                                                                                                                                                         |
  \| Organización                                   | Esta opción solamente está disponible si ha seleccionado Complemento.Opcional: Introduzca cualquier cadena de la organización necesaria para el complemento.  Consulte la documentación del complemento para determinar si este valor es necesario.                                                                                                                                                                                                                                                                                                                                                                                                                                         |
  \| Nombre de usuario                              | Esta opción solamente está disponible si ha seleccionado Complemento.Opcional: Introduzca cualquier nombre de usuario necesario para el servicio del complemento. Consulte la documentación del complemento para determinar si este valor es necesario.                                                                                                                                                                                                                                                                                                                                                                                                                                     |
  \| Contraseña                                     | Esta opción solamente está disponible si ha seleccionado Complemento.Opcional: Introduzca cualquier contraseña necesaria para el servicio del complemento. Consulte la documentación del complemento para determinar si este valor es necesario.                                                                                                                                                                                                                                                                                                                                                                                                                                            |
  \| Certificado                                    | Esta opción solamente está disponible si ha seleccionado Complemento.Opcional: Introduzca cualquier certificado necesario para que el servicio del complemento autentique al usuario. Consulte la documentación del complemento para determinar si este valor es necesario.                                                                                                                                                                                                                                                                                                                                                                                                                 |
  \| Tráfico Webkit del filtro                      | Esta opción solamente está disponible si ha seleccionado Complemento.Seleccione esta opción para incluir el tráfico Webkit en el filtro.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
  \| Tráfico del Socket del filtro                  | Esta opción solamente está disponible si ha seleccionado Complemento.Seleccione esta opción para incluir el tráfico del socket en el filtro.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
  \| Datos personalizados                           | Esta opción solamente está disponible si ha seleccionado Complemento.Opcional: Introduzca cualquier par clave/valor necesarios para el servicio del complemento. Consulte la documentación del complemento para determinar si este valor es necesario.                                                                                                                                                                                                                                                                                                                                                                                                                                      |
