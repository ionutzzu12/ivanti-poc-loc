Como administrador de Ivanti Neurons for MDM, puede configurar los ajustes del proxy de DNS mediante la Configuración de proxy de DNS para usuarios de dispositivos de iPhone y de iPad. Puede usar la carga útil del proxy DNS para especificar la aplicación que proporciona la extensión de red del proxy DNS y otros valores específicos del proveedor.

**Procedimiento**

::::WorkflowBlock
:::WorkflowBlockItem
Iniciar sesión en el portal administrativo de Ivanti Neurons for MDM.
:::

:::WorkflowBlockItem
Vaya a **Configuraciones**
:::

:::WorkflowBlockItem
Escriba **DNS** en el campo de búsqueda y haga clic en la **configuración del proxy de&#x20;****DNS**.
:::

:::WorkflowBlockItem
Introduzca un nombre y describa la configuración.
:::

:::WorkflowBlockItem
Introduzca los siguientes ajustes de Configuración del proxy de DNS:\* Identificador de paquetes de aplicaciones (Requerido).

- Identificador del grupo de proveedores
- Configuración del proveedor (Valor de la clave).
:::

:::WorkflowBlockItem
Haga clic en **Siguiente**.
:::

:::WorkflowBlockItem
Seleccione la opción **Habilitar esta configuración**.
:::

:::WorkflowBlockItem
Seleccione una de las siguientes opciones de distribución:
:::

:::WorkflowBlockItem
- Todos los dispositivos
- Ningún dispositivo (predeterminada)
- personalizada
  Haga clic en **Hecho**.
:::
::::
