手順

1. Ivanti Neurons for MDM にログインし、**\[管理]** > **\[Microsoft Azure]**&#x306B;移動します。
2. 左側のメニューで **\[Microsoft Azure]** > **\[デバイスコンプライアンス]** をクリックします。
3. \[アカウントを接続] の下で次の詳細情報を入力します。\* AzureテナントID - Microsoft Azureインスタンスを参照します。
   - 登録URL（任意） - デバイスがMDM登録されていない場合、この登録用URLがデバイスユーザーに表示されます。 設定時にはHTTPS形式を使用してください。 組織内でページをホストし、デバイスユーザーに登録情報を表示したい場合は、ここにリンクを追加します。
   - 修復URL（任意） - デバイスがコンプライアンス違反の場合、この修復用URLがデバイスユーザーに表示されます。 設定時にはHTTPS形式を使用してください。 組織内でページをホストし、デバイスユーザーに修復情報を表示したい場合は、ここにリンクを追加します。
4. \[アカウントを接続] をクリックします。 \[Azureテナント検証] ダイアログボックスが開きます。

現在Azureに接続されており、macOSデバイスのデバイスコンプライアンスの追加を希望するテナントは、アカウントの接続を切断した後、接続を再確立する必要があります。

1. \[Azureテナント検証] ダイアログで、手順1にあるリンクをクリックします。
2. ログインします。
3. 許可の内容を読み、\[許可する] をクリックします。

ログイン後のページで、再度ログインするよう求められた場合は、そのブラウザタブ/ウィンドウを閉じてください。

1. Ivanti Neurons for MDM に戻ります。 \[Azureアカウントの連携] ダイアログボックスで \[同意しました] チェックボックスを選択します。 \[確認] をクリックします。

- アカウントを編集するには \[アカウントを編集] をクリックします。
- アカウントの接続を解除するには \[アカウントを切断] をクリックします。 詳しい説明は[**Azureテナントのプロビジョニング解除**](./Azureテナントのプロビジョニング解除.md)を参照してください。
- アカウントの追加、編集、無効化のすべてのアクティビティはログに記録されます。
