Ivanti Neurons for MDM では、使用するペイロードに対する省エネルギー構成を有効にして、WindowsデバイスとmacOSデバイスの省エネルギー設定を構成できます。

**対象：**

- Windows 10以降、Ivanti Neurons for MDMがサポートする最新版まで。
- macOS 10.7以降、Ivanti Neurons for MDMがサポートする最新版まで。

**手順**

1. **\[構成]** > **\[+追加]** を開きます。
2. **\[省エネルギー]** 構成を検索して選択します。
3. \[OSを選択] セクションで、**\[Windows]** または **\[macOS]** を選択して **\[次へ]** をクリックします。
4. **\[+説明を追加]** をクリックし、構成の簡単な説明を入力します。
5. Windowsの場合、次の **\[省エネルギー]** 設定を構成します。\* [**デバイスはバッテリーで動作している**](./省エネルギー構成.md)
   - [**デバイスは電源につながれている**](./省エネルギー構成.md)
6. 次の **\[省エネルギー]** 設定を構成します。\* [**デスクトップのAC電源の省エネルギー設定**](./省エネルギー構成.md)
   - [**コンピューターの起動またはシャットダウンのスケジュール設定**](./省エネルギー構成.md)
   - [**ラップトップのAC電源の省エネルギー設定**](./省エネルギー構成.md)
   - [**ラップトップのバッテリー電源の省エネルギー設定**](./省エネルギー構成.md)
7. **\[次へ]** をクリックして配布設定を行います。
8. 配布オプションを1つ選択して **\[省エネルギー]** 構成を設定します。 配布オプションの構成についての詳細は、[**構成の操作**](./構成の操作.md)を参照してください。
9. **\[完了]** をクリックします。

## Windows省エネルギー構成

新しい省エネルギー構成がデバイスに適用された場合や、既存の省エネルギー構成の設定が変更された場合は、変更が適用されるよう、ユーザーが各自のデバイスを再起動する必要があります。

### デバイスはバッテリーで動作している

デバイスがバッテリーからの電力を使用している場合は、下表の省エネルギー設定を使用します。

| **設定**                    | **説明**                                                                                                                                                                                                                                    |
| ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **省エネルギーをオンにするバッテリーしきい値** | デバイスがバッテリー電力を使用している場合、省エネルギーをオンにするバッテリー充電レベルを入力します。0～100の値を入力できます。 バッテリー充電レベルを示すパーセンテージ値を入力します。 たとえば、80に設定すると、使用できるバッテリー残量が80%になるか、またはそれ以下のときに省エネルギーがオンになります。値を入力しないと、Ivanti Neurons for MDM はこの設定を変更も更新もしません。 デフォルトでは、Windowsが70%に設定します。 |
| **カバーを閉じたとき**             | - **何も実行しない**：デバイスは電源が入ったまま、バッテリー電力を使用し続けます。                                                                                                                                                                                              |

- **スリープ**：デバイスはスリープモードになり、少量のバッテリー電力を使用します。 コンピュータは電源が入ったまま、開いているアプリやファイルはランダムアクセスメモリ（RAM）に保存されます。
- **休止スリープ状態**：デバイスは休止モードになります。 開いているアプリやファイルはハードディスクに保存され、デバイスは電源が切れます。
- **シャットダウン**：デバイスはシャットダウンします。 開いているアプリやファイルは保存されずに閉じます。 |
  \| **電源ボタンが選択されたとき**         | \* **何も実行しない**：デバイスは電源が入ったまま、バッテリー電力を使用し続けます。
- **スリープ**：デバイスはスリープモードになり、少量のバッテリー電力を使用します。 コンピュータは電源が入ったまま、開いているアプリやファイルはRAMに保存されます。

## macOS省エネルギー構成

### デスクトップのAC電源の省エネルギー設定

以下の設定を使用して、macOSデスクトップデバイスのAC電源の省エネルギー設定を構成します。

| **設定**                   | **説明**                                                                                  |
| ------------------------ | --------------------------------------------------------------------------------------- |
| **\[名前]**                | この構成を識別する名前を入力します。                                                                      |
| **説明**                   | この構成の目的を明示する説明を入力します。                                                                   |
| **電源障害の後に自動的に起動する**      | 電源障害の後にデバイスが自動的に起動するようにします。                                                             |
| **ハードディスクをスリープにするまでの時間** | 1～180分の一定間隔でハードディスクがスリープモードに入るようにするには、このチェックボックスを有効にします。値を0分にすると、ハードディスクはスリープモードに入りません。 |
| **ディスプレイをオフにするまでの時間**    | 1～180分の一定間隔でディスプレイの電源を切るには、このチェックボックスを有効にします。値を0分にすると、ディスクプレイの電源は切れません。                 |
| **ダイナミック パワー ステップ**      | ダイナミックパワーステップを有効にするには、このチェックボックスを選択します。                                                 |
| **プロセッサ動作周波数を下げる**       | プロセッサの処理速度を下げるには、このチェックボックスを選択します。                                                      |
| **コンピューターをスリープにするまでの時間** | 1～180分の一定間隔でコンピューターがスリープモードに入るようにするには、このチェックボックスを有効にします。値を0分にすると、コンピューターはスリープモードに入りません。 |
| **ネットワークアクセスによるスリープ解除**  | ネットワークアクセスがあったときにコンピューターのスリープを解除するには、このチェックボックスを選択します。                                  |
| **モデムリングによるスリープ解除**      | モデムに着信があったときにコンピューターのスリープを解除するには、このチェックボックスを選択します。                                      |

### コンピューターの起動またはシャットダウンのスケジュール設定

以下の設定を使用して、macOSデバイスの起動またはシャットダウンのスケジュール設定を構成します。

### デバイスの電源オフの設定

一定間隔でデバイスをシャットダウンするには、**\[デバイスの電源を切る]** チェックボックスを選択します。

| **設定**      | **説明**                                |
| ----------- | ------------------------------------- |
| **イベントタイプ** | ドロップダウンリストから次のイベントタイプを1つ選択します。\* スリープ |

- 再起動
- シャットダウン |
  \| **条件**      | イベントタイプをスケジュールする曜日を1つ以上選択します。                        |
  \| **タイミング**   | イベントタイプをスケジュールする時刻を選択します。                            |

### デバイスの起動またはスリープ解除の設定

一定間隔でデバイスをスリープ解除するには、**\[デバイスを起動またはスリープ解除]** チェックボックスを選択します。

| **設定**      | **説明**                                   |
| ----------- | ---------------------------------------- |
| **イベントタイプ** | ドロップダウンリストから次のイベントタイプを1つ選択します。\* ウェイクアップ |

- 起動 / スリープ解除 |
  \| **条件**      | イベントタイプをスケジュールする曜日を1つ以上選択します。                         |
  \| **タイミング**   | イベントタイプをスケジュールする時刻を選択します。                             |
