手順

1. Microsoft Endpoint Manager（[**https://endpoint.microsoft.com**](https://endpoint.microsoft.com/)）にログインします。
2. Microsoft Endpoint Manager管理センターページで \[ホーム] > \[エンドポイント セキュリティ] > \[条件付きアクセス] を開きます。
3. \[ポリシー] をクリックし、\[+新しいポリシー] をクリックします。

::Image[]{src="Resources/Images/MSEM_01_AddnewPol.png" signedSrc="Resources/Images/MSEM_01_AddnewPol.png" size="27" caption alt isUploading="false" sha initialPath="Resources/Images/MSEM_01_AddnewPol.png" githubPath="Resources/Images/MSEM_01_AddnewPol.png" position="flex-start" indent="2"}

4. 条件付きアクセスポリシーの名前を入力します。

::Image[]{src="Resources/Images/MSEM_02_AddnewPol.png" signedSrc="Resources/Images/MSEM_02_AddnewPol.png" size="23" caption alt isUploading="false" sha initialPath="Resources/Images/MSEM_02_AddnewPol.png" githubPath="Resources/Images/MSEM_02_AddnewPol.png" position="flex-start" indent="2"}

5. \[割り当て] をクリックし、ユーザーやグループにポリシーを割り当てます。

::Image[]{src="Resources/Images/MSEM_03_AddnewPol.png" signedSrc="Resources/Images/MSEM_03_AddnewPol.png" size="21" caption alt isUploading="false" sha initialPath="Resources/Images/MSEM_03_AddnewPol.png" githubPath="Resources/Images/MSEM_03_AddnewPol.png" position="flex-start" indent="2"}

6. \[クラウド アプリまたは操作] をクリックし、\[選択] をクリックします。 保護する必要のあるアプリを検索し、選択します。

::Image[]{src="Resources/Images/MSEM_04_AddnewPol.png" signedSrc="Resources/Images/MSEM_04_AddnewPol.png" size="28" caption alt isUploading="false" sha initialPath="Resources/Images/MSEM_04_AddnewPol.png" githubPath="Resources/Images/MSEM_04_AddnewPol.png" position="flex-start" indent="2"}

7. 条件をクリックし、\[デバイス プラットフォーム] をクリックします。 適切なデバイスプラットフォームを選択します。

::Image[]{src="Resources/Images/MSEM_05_AddnewPol.png" signedSrc="Resources/Images/MSEM_05_AddnewPol.png" size="22" caption alt isUploading="false" sha initialPath="Resources/Images/MSEM_05_AddnewPol.png" githubPath="Resources/Images/MSEM_05_AddnewPol.png" position="flex-start" indent="2"}

8. 新しい条件付きアクセスポリシーのページから \[アクセス制御] を開き、\[許可] をクリックして許可またはブロックを選択します。

::Image[]{src="Resources/Images/MSEM_06_AddnewPol.png" signedSrc="Resources/Images/MSEM_06_AddnewPol.png" size="10" caption alt isUploading="false" sha initialPath="Resources/Images/MSEM_06_AddnewPol.png" githubPath="Resources/Images/MSEM_06_AddnewPol.png" position="flex-start" indent="2"}

9. 新しいポリシーを有効にするには \[オン] をクリックします。

::Image[]{src="Resources/Images/MSEM_07_AddnewPol.png" signedSrc="Resources/Images/MSEM_07_AddnewPol.png" size="27" caption alt isUploading="false" sha initialPath="Resources/Images/MSEM_07_AddnewPol.png" githubPath="Resources/Images/MSEM_07_AddnewPol.png" position="flex-start" indent="2"}

10. \[作成] をクリックします。
