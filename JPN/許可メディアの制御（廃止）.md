許可メディアの制御構成では、macOSにおけるログアウト時のさまざまな物理メディアのマウント、アンマウントおよび取り出しを管理します。

次のオプションを設定できます。

| 設定                                                                  | 操作内容                  |
| ------------------------------------------------------------------- | --------------------- |
| 名前                                                                  | この構成を識別する名前に入力します。    |
| 説明                                                                  | この構成の目的を明示する説明を入力します。 |
| **構成設定**                                                            |                       |
| **メディアタイプごとのマウント制御**                                                |                       |
| 各メディアタイプのマウント制御をオンにし、マウント設定を実行します。 マウント制御をオフにすると、OSのデフォルト設定が適用されます。 |                       |
| **メディアタイプ**                                                         | **マウント設定**            |
| - CD                                                                |                       |

- BD                                                                                                  | \* 認証により読み取り専用

- ネットワークディスク                                 | \* 読み取り専用
- マウントを拒否

- 認証 |

  \| 各メディアタイプのアンマウント制御をオンにし、アンマウント設定を実行します。 アンマウント制御をオフにすると、OSのデフォルト設定が適用されます。 各メディアタイプに \[マウントを拒否] を設定する場合は注意してください。 |                                     |

## 構成の配布

**手順**

::::WorkflowBlock
:::WorkflowBlockItem
上記の表を使用してオプションを設定します。
:::

:::WorkflowBlockItem
**\[次へ]** をクリックします。
:::

:::WorkflowBlockItem
**\[この構成を有効化]** オプションを選択します。
:::

:::WorkflowBlockItem
以下の配布オプションから1つ選択します。
:::

:::WorkflowBlockItem
- すべてのデバイス
- デバイスなし（デフォルト）
- カスタム
  **\[完了]** をクリックします。
:::
::::
