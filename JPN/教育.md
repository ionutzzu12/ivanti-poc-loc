**ライセンス：&#x20;**&#x47;old

**対象：**&#x76E3;視対象のiOS 9.3+

リーダーおよびメンバー向けのApple EducationペイロードとClassroomアプリを構成します。 次の表はEducationの設定を示します。

| **設定** | 操作内容                   |
| ------ | ---------------------- |
| 名前     | この構成を識別する名前に入力します。     |
| 説明     | この構成の目的を明示する説明を入力します。  |
| 構成の種類  | 次の種類から1つを選択します。\* リーダー |

- メンバー                                              |
  \| この構成を有効化 | - 選択したデバイスにこの構成を適用する場合に選択します。
- 選択を解除すると、過去に適用されていたすべてのデバイスからこの構成が削除されます。 |
  \| 配布する     | 以下の配布オプションから1つ選択します。- すべてのデバイス
- デバイスなし
- カスタム                            |
