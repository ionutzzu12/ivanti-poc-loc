iOSのカスタム構成により、AppleのiPhone構成ユーティリティなど、別のアプリによって作成されたiOS構成プロファイルをアップロードし配信することができるようになります。

## iOSカスタム設定

| **設定**  | 操作内容                                                    |
| ------- | ------------------------------------------------------- |
| 名前      | この構成を識別する名前に入力します。                                      |
| 説明      | この構成の目的を明示する説明を入力します。                                   |
| ファイルデータ | 構成ファイルをドラッグするか、**\[ファイルを選択する]** をクリックしてファイルシステムから選択します。 |
