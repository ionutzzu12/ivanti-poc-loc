Ivanti Neurons for MDM enables the creation of Software Update Enforcement Settings and defines the behavior of updates to the end user for cadence and deferrals.

Applicable to:

- iOS 18.0 Supervised through the most recently released version as supported by Ivanti Neurons for MDM.
- macOS 15.0 Supervised through the most recently released version as supported by Ivanti Neurons for MDM.

**Procedure**

1. Go to **Configurations** > **+Add**.
2. Search and select the **Software Update Settings** configuration.
3. Configure the **Software Update Settings** settings as per the following table:| **Setting**                                        | **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
   \| -------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   \| Name                                               | Enter a name that identifies this configuration.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
   \| Description                                        | Enter a description that clarifies the purpose of this configuration.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
   \| **iOS 18+ Supervised** and **macOS15+**            |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
   \| Beta Enrollment                                    |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
   \| Program Enrollment (Optional)                      | Select one of the following options:\* **Allowed**: The user can enroll in any applicable beta programs associated with their logged in Apple Account.
   - **Always On**: The beta programs specified by the organization are used, and the user is not able to enroll in a beta program using their logged in Apple Account.**Offer Program** - With Offer Program, you can enforce a required program. When enforcing, you can set the required versions for iOS and macOS devices.**Require Program** - The device automatically enrolls in this beta program.
   - **Always Off**: The device is not allowed to enroll in any beta programs. The system removes the device from any beta programs, if already enrolled. |
     \| **iOS 18+ Supervised** and **macOS15+ Supervised** |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
     \| Notifications                                      | Select the **Notifications** checkbox and the **Show all software update enforcement notifications** checkbox to display all the software update enforcement notifications in the user's device.If you do not select the check boxes, the device displays the notifications triggered one hour before the enforcement deadline and restarts the countdown notifications.                                                                                                                                                                                                                                                                                                                                              |
     \| Automatic Actions                                  |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
     \| Download                                           | Select one of the following options:\* **Allowed**: The user can turn “On” or “Off” for automatic downloads.
   - **Always On**: The automatic downloads are always “On” for the user.
   - **Always Off**: The automatic downloads are always “Off” for the user.                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
     \| Install OS Updates                                 | Select one of the following options:\* **Allowed**: The user can turn “On” or “Off” to automatically install OS updates.
   - **Always On**: The automatic installation of OS updates is always “On” for the user.
   - **Always Off**: The automatic installation of OS updates is always “Off” for the user.                                                                                                                                                                                                                                                                                                                                                                                                               |
     \| Rapid Security Response                            |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
     \| Rapid Security Response                            | Select the **Rapid Security Response** checkbox to control the installation and rollback of updates. Select the following options:\* **Enable**: If selected, it offers **Rapid Security Response** installation to the user.
   - **Enable Rollback**: If selected, it offers **Rapid Security Response** rollback to the user.                                                                                                                                                                                                                                                                                                                                                                                          |
     \| **iOS 18+ Supervised**                             |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
     \| Recommended Cadence                                | Select one of the following options to show the software updates to the user when more than one update is available:\* **All**: Shows all software update versions.
   - **Oldest**: Shows only the oldest (lower numbered) software update version.
   - **Newest**: Shows only the newest (highest-numbered) software update version.                                                                                                                                                                                                                                                                                                                                                                                      |
     \| Deferrals                                          | You can choose to postpone the software updates for all supervised devices.You cannot postpone the **Rapid Security Responses**.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
     \| Combined Period In Days                            | Set the days to postpone a major or minor system update on the device. Once configured, the major or minor software updates will appear after a specified number of days following their release.You can set any values between 1 and 90 days.                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
     \| **macOS 15+ Supervised**                           |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
     \| Allow Standard User OS Updates                     | Select the **Allow Standard User OS Updates** checkbox to allow a user to perform OS updates.Select the **Allow standard user to perform Major and Minor Software Updates** checkbox to allow a standard user to perform minor and major software updates.Only the administrators can perform minor and major software updates if you do not select the checkbox.                                                                                                                                                                                                                                                                                                                                                     |
     \| Deferrals                                          |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
     \| Major Update Period In Days                        | Set the days to postpone a major OS software update on the device. Once configured, the major software update will appear after a specified number of days following its release.You can set any values between 1 and 90 days.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
     \| Minor Update Period In Days                        | Set the days to postpone a minor OS software update on the device. Once configured, the major software update will appear after a specified number of days following its release.You can set any values between 1 and 90 days.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
     \| System Update Period In Days                       | Set the days to postpone a system OS update on the device. Once configured, the major software update will appear after a specified number of days following its release.You can set any values between 1 and 90 days.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
     \| Automatic Actions                                  |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
     \| Install Security Update                            | Select one of the following options:\* **Allowed**: The user can turn “On” or “Off” the installation of security updates.
   - **Always On**: The installation of security updates is always “On” for the user.
   - **Always Off**: The installation of security updates is always “Off” for the user.                                                                                                                                                                                                                                                                                                                                                                                                                      |
4. Click **Next** to configure the distribution settings.
5. Select one of the distribution options to set up the **Software Update Settings** configuration. For more information about configuring distribution options, see [**Working with Configurations**](./Working_with_Configurations.md).
6. Click **Done**.
