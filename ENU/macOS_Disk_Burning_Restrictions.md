macOS Disk Burning Restrictions manage disk burning restrictions in macOS. You can configure the [**macOS Finder Settings**](./macOS_Finder_Settings.md) to enable or disable disc burning options from the Finder app on macOS.

You can set the following options:

| Setting                 | What To Do                                                            |
| ----------------------- | --------------------------------------------------------------------- |
| Name                    | Enter a name that identifies this configuration.                      |
| Description             | Enter a description that clarifies the purpose of this configuration. |
| **Configuration Setup** |                                                                       |
| **Setting**             | **What To Do**                                                        |
| Allow Disk Burning      | - ON                                                                  |

- OFF
- Require Authentication                                   |
