**Applicable to:** macOS 11.5+

The administrator can set or change recovery lock for device reboot for macOS device running on Apple silicon. A recovery lock prevents starting up of the macOS devices in recovery mode, unless the passcode is entered.

**Procedure**:

::::WorkflowBlock
:::WorkflowBlockItem
Go to **Devices**.
:::

:::WorkflowBlockItem
To set or change the recovery lock for reboot:
:::

:::WorkflowBlockItem
1. Click the user display name the device is associated with to view the device details page. Perform one of the following steps:
2. In the **Overview** section, expand **Recovery Lock** and click **Set Password** or click **Change Password**. Or Click the **Actions** ellipsis and click **Set/Change Recovery Lock**.
3. In the **Set/Change Recovery Lock** dialogue box do the following:1) **Current Password**: Enter the current password here. Keep it empty if you are setting it for the first time.
   2\) **Password**: Enter the password you want to set.
   3\) **Confirm Password**:Â Re-type the password you want to set.
   Click **Set/Change Recovery Lock**.
:::
::::

In Overview, **Recovery Lock Enabled** shows the status of the Recovery Lock passcode.

Administrators can also clear the passcode by removing the existing passcode and click **Set/Change Recovery Lock**.
