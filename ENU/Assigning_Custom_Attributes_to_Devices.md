You can assign custom device attributes such asÂ internal ID to one or more devices. Each attribute has a corresponding value that you can use for tasks like creating configurations and device groups. After you create custom attributes, you can assign them to devices. For more information about managing attributes, see [**Attributes**](./Attributes.md).

**Procedure**

::::WorkflowBlock
:::WorkflowBlockItem
Log in to the Administrative portal.
:::

:::WorkflowBlockItem
Go to **Devices**.
:::

:::WorkflowBlockItem
Select one or more devices.
:::

:::WorkflowBlockItem
Click **Actions**.
:::

:::WorkflowBlockItem
Select **Assign Custom Attributes**.
:::

:::WorkflowBlockItem
Select *one* of the following options:
:::

:::WorkflowBlockItem
- Force assign (overwrite) all attributes even if any existing values are found.
- Overwrite only if value is empty, and skip attributes with existing values.
  Select the attributes you want to assign and enter their values (empty values are not allowed).
:::

:::WorkflowBlockItem
Click **Assign**.
:::
::::

The **Custom Device Attributes** with their values can be exported to CSV format from the **Device Details** page.
